Задание 1

1) Используйте команды операционной системы Linux и создайте
две новых директории – «Игрушки для школьников» и «Игрушки
для дошколят»
mkdir Игрушки\ для\ школьников
mkdir "Игрушки для дошколят"

2) Создайте в директории «Игрушки для школьников» текстовые
файлы - «Роботы», «Конструктор», «Настольные игры»

cd Игрушки\ для\ школьников/
touch Роботы.txt Конструктор.txt Настольные\ игры.txt

3) Создайте в директории «Игрушки для дошколят» текстовые
файлы «Мягкие игрушки», «Куклы», «Машинки»

cd ../Игрушки\ для\ дошколят/
touch Мягкие\ игрушки.txt Куклы.txt Машинки.txt

4) Объединить 2 директории в одну «Имя Игрушки»

cd ~
mkdir Имя\ Игрушки
cp -r Игрушки\ для\ дошколят/* Имя\ Игрушки
cp -r Игрушки\ для\ школьников/* Имя\ Игрушки

5) Переименовать директорию «Имя Игрушки» в «Игрушки»

mv Имя\ Игрушки Игрушки

6) Просмотреть содержимое каталога «Игрушки».

# если будем находится в той же директории что и «Игрушки»
ll Игрушки/

# если из другой директории, то можно так
ll ~/Игрушки/

7) Установить и удалить snap-пакет. (Любой, какой хотите)

# установим Postman
# проверим не установлен ли он у нас
snap list postman
# найдем такой пакет
snap search postman
# установим пакет
sudo snap install postman
# еще раз проверим установленные пакеты
snap list
# запустим, чтобы проверить что всё установилось
postman

# удалим Postman
sudo snap remove postman

8) Добавить произвольную задачу для выполнения каждые 3
минуты (Например, запись в текстовый файл чего-то или
копирование из каталога А в каталог Б).

# настроим автоматический бэкап файла c важной информацией info.txt
# добавим правило для планировщика задач cron
crontab -e
# добавили строку - каждые 3 минуты перенаправляем поток вывода из файла info.txt в файл info.txt.back
*/3 * * * * cat /home/eugenepro/info.txt > /home/eugenepro/info.txt.back
либо можно воспользоваться вторым вариантом:
# утилитой cp (копирование файла)
*/3  * * * * cp –b /home/eugenepro/info.txt /home/eugenepro/info.txt.bak
